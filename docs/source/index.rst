Welcome to github-pages-test's documentation!
=============================================

github-pages-test is a Python library to test documentation creation. It pulls data from the `Open Food
Facts database <https://world.openfoodfacts.org/>`_ and offers a *simple* and
*intuitive* API.

.. note::

   This project is under active development. The documentation may be incomplete or incorrect.
